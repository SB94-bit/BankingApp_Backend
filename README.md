BankingApp_Backend
==================================== ENG ==================================== 
Simulated backend made with Springboot of a Banking app that allows users to sign in, create and account and transfer money. 
The app was designed to use the schema banking_system_db in Mysql (port: 3306) 
The database management is handled by hibernate. 
Authentication process handled by Spring security and token Jwt. 
Screenshots available in the project.

==================================== ITA ==================================== 
Backend di una Banking app realizzato con Springboot. 
L'app è configurata per comunicare con lo schema banking_system_db di Mysql (porta: 3306). 
Il db è gestito da hibernate. 
Gestione dell'autenticazione con Spring security e token Jwt. 
Possibilità di registrarsi come user, aprire più conti e trasferire denaro. 
Screenshots disponibili dentro la cartella del progetto
